----------------------------
Notas e Restrições
----------------------------
  • Ao utilizar o Serviço de Terminal no SO do servidor, verifique as Ferramentas do Utilitário de Configuração com o modo de instalação.
    O método para executar o modo de instalação é o seguinte.
    - Abra [Adicionar ou Remover Programas] no [Painel de Controlo] e depois execute o As Ferramentas do Utilitário de Configuração.

  • Durante o ecrã é exibido o ecrã [Controlo de Contas de Utilizador] dependendo da permissão do utilizador que executou as ferramentas. Nesse caso, clique em [Permitir] ou insira a palavra-passe do administrador e clique em [OK]. Além disso, é apresentado o ecrã [Assistente de Compatibilidade do Programa] ao sair das ferramentas. Quando [Este programa estiver instalado corretamente.] é selecionado, este ecrã não será apresentado da próxima vez.

  • Seleção da fila NDS
    A impressão não pode ser realizada se uma porta/fila na árvore NDS que não existe na porta de destino for selecionada no Setup Disk Creation Tool.

  • Quando "Ocorreu um erro durante a atualização da versão" é apresentado durante a atualização da versão, elimine o controlador de impressão e crie uma nova impressora após reiniciar.

  • Reiniciar o sistema de reinicie durante a instalação/atualização de versão
    Ao instalar o controlador de impressão ou atualizar a versão do controlador, reinicie sempre o sistema seguindo as instruções fornecidas na mensagem exibida, antes de utilizar o controlador de impressão. Podem ocorrer erros inesperados se o sistema não for reiniciado.

  • Refletir o tamanho do documento usando o Setup Tool
    Quando um papel exclusivo do controlador é selecionado para o tamanho do documento, o tamanho do documento não será refletido corretamente.

  • Se for exibida a mensagem "As definições da impressora atual podem ser alteradas se instalar o novo controlador. Deseja continuar?" ao instalar o controlador com o Setup Tool, selecione a caixa de verificação "Atualizar o controlador utilizando o disco de configuração" no Setup Disk Creation Tool e crie um disco de configuração.

  • Se o Setup Tool for criada a partir de uma disquete utilizando o Setup Disk Creation Tool com a opção "Atualizar o controlador utilizando o disco de configuração" selecionada, não é possível executar o verup.exe na disquete. Por favor execute setup.exe. Ao executar setup.exe, o controlador de impressão também será atualizado.

  • Método para especificar a porta para a impressora de rede
    A porta da impressora de rede não pode ser especificada a partir do ecrã [Procurar impressora] no Setup Disk Creation Tool. Introduza o nome da porta diretamente no [Caminho da rede da impressora] no ecrã [Adicionar Porta].

  • Refletir as definições de impressão para Setup Tool
    Quando executar o Setup Tool com a opção [Permitir que os utilizadores gerais executem a configuração] especificada, as definições de impressão guardadas no disco de configuração serão refletidas nas definições de impressão da conta do administrador. Além disso, mesmo quando executar o Setup Tool sem a opção [Permitir que os utilizadores gerais executem a configuração] especificada, mas com a conta de administrador especificada no ecrã [Controlo de Contas de Utilizador], as definições de impressão guardadas no disco de arranque serão refletidas nas definições de impressão da conta do administrador+. Para refletir as definições de impressão de todos os utilizadores, incluindo o utilizador que executou o Setup Tool, configurar [Impressão Padrão] da impressora ao criar o disco de configuração. Pode definir [Predefinições de impressão] a partir do separador [Avançado] do janela de diálogo Propriedades da Impressora.

----------------------------
Opmerkingen en beperkingen
----------------------------
  • Als u Terminal Service gebruikt in het server-besturingssysteem, voert u Setup Utility Tools uit in de installatiemodus.
    De methode om de installatiemodus uit te voeren is als volgt.
    - Open [Programma's toevoegen of verwijderen] in [Configuratiescherm] en voer vervolgens Setup Utility Tools uit.

  • Tijdens het opstarten wordt het scherm [Gebruikersaccountbeheer] weergegeven, afhankelijk van de toestemming van de gebruiker die de tools heeft uitgevoerd. Klik in dat geval op [Toestaan] of voer het beheerderswachtwoord in en klik op [OK]. Bovendien wordt het scherm [Assistent voor programmacompatibiliteit] weergegeven wanneer u de hulpprogramma's afsluit. Als [Dit programma is correct geïnstalleerd.] is geselecteerd, wordt dit scherm de volgende keer niet weergegeven.

  • NDS-wachtrij selecteren
    Afdrukken kan niet worden uitgevoerd als een poort/wachtrij in de NDS-structuur die niet bestaat in de doelpoort is geselecteerd in de Setup Disk Creation Tool.

  • Wanneer "Er is een fout opgetreden tijdens de versie-upgrade" wordt weergegeven tijdens de versie-upgrade, verwijder dan de printerdriver en maak een nieuwe printer nadat deze opnieuw is opgestart.

  • Systeem herstarten tijdens installatie/versie-upgrade
    Wanneer u de printerdriver installeert of de driverversie bijwerkt, start u het systeem altijd opnieuw op volgens de instructies in het weergegeven bericht, voordat u de printerdriver gebruikt. Er kunnen onverwachte fouten optreden als het systeem niet opnieuw wordt opgestart.

  • Documentgrootte weergeven met behulp van Setup Tool
    Wanneer een papier dat uniek is voor de driver is geselecteerd voor documentformaat, wordt het documentformaat niet correct weergegeven.

  • Als "De huidige printerinstellingen kunnen worden gewijzigd als u de nieuwe driver installeert. Wilt u doorgaan?" wordt weergegeven wanneer u de driver installeert met de Setup Tool, selecteert u het selectievakje "Driver upgraden met installatieschijf" in de Setup Disk Creation Tool en maakt u een installatieschijf.

  • Als Setup Tool is gemaakt vanaf een diskette met behulp van Setup Disk Creation Tool met "Driver upgraden met installatieschijf" geselecteerd, kunt u verup.exe niet uitvoeren op de diskette. Voer setup.exe uit. Door setup.exe uit te voeren, wordt ook de printerdriver geüpgraded.

  • Methode om poort op te geven voor netwerkprinter
    De netwerkprinterpoort kan niet worden opgegeven vanaf het scherm [Bladeren naar printer] in Setup Disk Creation Tool. Voer de poortnaam rechtstreeks in [Netwerkpad van printer] op het scherm [Poort toevoegen] in.

  • Afdrukinstellingen weergeven voor Setup Tool
    Wanneer u de Setup Tool uitvoert met [Algemene gebruikers toestaan installatie uit te voeren] opgegeven, worden de afdrukinstellingen die zijn opgeslagen op de installatieschijf weergegeven in de afdrukinstellingen van het beheerdersaccount. Bovendien, zelfs wanneer u de Setup Tool uitvoert zonder [Algemene gebruikers toestaan setup uit te voeren] opgegeven maar met het beheerdersaccount opgegeven op het [Gebruikersaccountbeheer]-scherm, worden de afdrukinstellingen die zijn opgeslagen op de installatieschijf weergegeven in de afdrukinstellingen van het beheerdersaccount. Om de instellingen weer te geven in de afdrukinstellingen van alle gebruikers, inclusief de gebruiker die de Setup Tool heeft uitgevoerd, configureert u [Standaardinstellingen printer] van de printer bij het maken van de installatieschijf. U kunt [Standaardinstellingen printer] instellen op het tabblad [Geavanceerd] van het dialoogvenster Printereigenschappen.

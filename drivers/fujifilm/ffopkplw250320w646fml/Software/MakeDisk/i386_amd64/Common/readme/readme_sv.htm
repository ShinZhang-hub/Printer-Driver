----------------------------
Anteckningar och begränsningar
----------------------------
  • När du använder Terminal Service i serverns OS, kör Setup Utility Tools i installationsläge.
    Metoden för att köra installationsläget är följande.
    - Öppna [Lägg till eller ta bort program] i [Kontrollpanelen] och kör sedan Setup Utility Tools.

  • Under uppstart visas skärmen [User Account Control] beroende på behörigheten för användaren som körde verktygen. Klicka i så fall på [Tillåt] eller ange administratörslösenordet och klicka på [OK]. Dessutom visas skärmen [Assistenten för programkompatibilitet] när du avslutar verktygen. När [Det här programmet är korrekt installerat.] är valt visas inte den här skärmen nästa gång.

  • Välja NDS-kö
    Det går inte att skriva ut om en port/kö i NDS-trädet som inte finns i målporten har valts i Setup Disk Creation Tool.

  • När "Ett fel har inträffat under uppgraderingen" visas under en uppgradering, tar du bort skrivardrivrutinen och lägger till en ny skrivare efter omstart.

  • Omstart av systemet under Installation/Uppgradering
    När du installerar skrivardrivrutinen eller uppgraderar drivrutinsversionen ska du alltid starta om systemet enligt instruktionerna i meddelandet som visas innan du använder skrivardrivrutinen. Oväntade fel kan uppstå om systemet inte startas om.

  • Återspegla dokumentstorlek med hjälp av Setup Tool
    När ett papper som är unikt för drivrutinen väljs för dokumentstorlek kommer dokumentstorleken inte att återspeglas korrekt.

  • Om "Aktuella skrivarinställningar kan ändras om du installerar den nya drivrutinen. Vill du fortsätta?" visas när du installerar drivrutinen med Setup Tool, markerar du kryssrutan "Uppgradera drivrutinen med installationsdisk" i Setup Disk Creation Tool och skapar en installationsdisk.

  • Om Setup Tool skapas från en diskett med hjälp av Setup Disk Creation Tool med "Uppgradera drivrutin med installationsdisk" markerat kan du inte köra verup.exe på disketten. Vänligen kör setup.exe. Genom att köra setup.exe uppgraderas även drivrutinen för skrivaren.

  • Tillvägagångssätt för att ange port för nätverksskrivare
    Porten för nätverksskrivare kan inte anges från skärmen [Bläddra efter skrivare] i Setup Disk Creation Tool. Ange portnamnet direkt i [Skrivarens nätverkssökväg] på skärmen [Lägg till port].

  • Återspegla utskriftsinställningar för Setup Tool
    När du kör Setup Tool med [Tillåt standardanvändare att köra installationsprogrammet] angivet återspeglas utskriftsinställningarna som sparats på installationsdisken i utskriftsinställningarna för administratörskontot. Utöver det, även när du kör Setup Tool utan [Tillåt standardanvändare att köra installationsprogrammet] angivet men med ett administratörskonto angivet vid skärmen för [User Account Control] återspeglas utskriftsinställningarna som sparats på installationsdisken i utskriftsinställningarna för administratörskontot. För att återspegla inställningarna under utskriftsinställningar för alla användare, inklusive användaren som körde Setup Tool, konfigurerar du [Standardvärden] för skrivaren när du skapar installationsdisken. Du kan ställa in [Standardvärden] på fliken [Avancerat] i dialogrutan Egenskaper för skrivare.

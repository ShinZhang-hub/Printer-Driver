----------------------------
Примечания и ограничения
----------------------------
  • При использовании службы терминалов в ОС сервера запустите Setup Utility Tools в режиме установки.
    Способ запуска режима установки выглядит следующим образом.
    - Откройте [Установка и удаление программ] в [Панель управления] и запустите Setup Utility Tools.

  • Во время запуска отображается меню [Управление учетными записями] в зависимости от разрешения пользователя, который запустил инструменты. В этом случае нажмите [Разрешить] или введите пароль администратора и нажмите [OK]. Кроме того, при выходе из инструментов отображается меню [Помощник по совместимости программ]. Если выбран параметр [Эта программа установлена правильно.], это меню не будет отображаться в следующий раз.

  • Выбор очереди NDS
    Печать невозможна, если в Setup Disk Creation Tool выбран порт/очередь в дереве NDS, который не существует в целевом порту.

  • Если во время обновления версии отображается сообщение "Произошла ошибка во время обновления версии", удалите драйвер печати и создайте новый принтер после перезагрузки.

  • Перезагрузка системы во время установки/обновления версии
    При установке драйвера печати или обновлении версии драйвера всегда перезагружайте систему, следуя инструкциям, приведенным в появившемся сообщении, перед использованием драйвера печати. Если система не перезагрузится, могут возникнуть непредвиденные ошибки.

  • Отображение размера документа с помощью Setup Tool
    Если для размера документа выбрана бумага, уникальная для драйвера, размер документа будет отражен неправильно.

  • Если "Текущие настройки принтера могут быть изменены при установке нового драйвера. Хотите продолжить?" отображается при установке драйвера с помощью Setup Tool, установите флажок "Обновить драйвер с помощью установочного диска" в Setup Disk Creation Tool и создайте установочный диск.

  • Если Setup Tool создано с дискеты с помощью Setup Disk Creation Tool с выбранным параметром "Обновить драйвер с помощью установочного диска", запустить verup.exe с дискеты невозможно. Запустите setup.exe. При запуске setup.exe драйвер печати также будет обновлен.

  • Способ указания порта для сетевого принтера
    Порт сетевого принтера не может быть указан в меню [Поиск принтера] в Setup Disk Creation Tool. Введите имя порта непосредственно в [Сетевой путь принтера] в меню [Добавить порт].

  • Отображение настроек печати для Setup Tool
    При запуске Setup Tool с заданным параметром [Разрешить обычным пользователям запускать программу установки] настройки печати, сохраненные на установочном диске, будут отражены в настройках печати учетной записи администратора. Кроме того, даже при запуске Setup Tool без заданного параметра [Разрешить обычным пользователям запускать программу установки], но с учетной записью администратора, указанной в меню [Управление учетными записями пользователей], настройки печати, сохраненные на установочном диске, будут отражены в настройках печати учетной записи администратора. Чтобы отразить настройки печати всех пользователей, включая пользователя, запустившего Setup Tool, задайте [Настройки печати по умолчанию] принтера при создании установочного диска. [Настройки печати по умолчанию] можно задать на вкладке [Дополнительно] диалогового окна "Свойства принтера".

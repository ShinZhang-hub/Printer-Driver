
================================================================================
ART EX Print Driver Improvements in v7.0.x
================================================================================

The improvements made so far for each model related to this driver are described.


- Apeos C8180/C7580/C6580

[Improvements from Ver 7.0.0]
* Corrected wrong UI terms.
* Removed unnecessary buttons and options.
* Fixed the issue that Hole Punch cannot be selected for a certain paper size when using
the printer installed with Finisher Type C4.

[Improvements from Ver 7.0.1]
* Finisher Type C5 is now supported. If you want to
use Staple-Free Staple function, Finisher Type C5 is required separately.
* Added the ability to control [Skip Blank Pages] function for 2 Sided Printing.
* Added the function to print images on the overlap margin area
when printing Poster.

[Improvements from Ver 7.0.4]
* Fixed the printing problem occurred when the Plain Side 2 was selected as
  [Paper Type for Auto Tray].
* Corrected the direction of Z Fold Half Sheet icon in the case of Print Mixed
  Size Originals setting.

[Improvements from Ver 7.0.6]
* Fixed the problem that [Use Account Management Tool] of the [Accounting System]
  function cannot be selected when using the Job Owner Setup tool.

[Improvements from Ver 7.0.7]
* Added Trustmark Feature.
* Fixed a symptom that the software terminates abnormally when ""Get Information 
  from Printer"" is issued under a certain condition.

[Improvements from Ver 7.0.8]
* Fixed an issue that an application may freeze when printing from the application
  after opening properties of this driver on Windows 11 Version 22H2 or later.

[Improvements from Ver 7.0.9]
* Improved behavior of Saved Settings function.

[Improvements from Ver 7.0.10]
* Fixed an issue that occasionally caused the settings of our already
  installed printer drivers and direct fax drivers to be reset to their
  default values when installing a new printer driver.
